Constraints
    NFFG
        [x] the name uniquely identify the nffg in the system
        []

    NODE
        [x] the name uniquely identify the node in the system
        [x] the associatedNffg must correspond to the parent nffg
        [x] the hostingHost must exist in the system
        [x] the functionalityType must exist in the catalogue (VNF)
        []

    VNF
        [x] the name uniquely identify the VNF in the system
        []

    LINK
        [x] the name uniquely identify the link within the nffg
        [x] the sourceNode must correspond to belonging Node
        [x] the destinationNode must exist within the NFFG
      ? [x] the (sourceNode,destinationNode) pair must not be repeated
           within the node even if the link has a different name
        []

    HOST
        [x] the name uniquely identify the host in the system
        [x] the nodes in allocatedNodes must not repeat and
            the same node cannot be hosted on 2 hosts
        [x] the nodes must exist within the deployedNFFGs
        []

    CONNECTION
        [x] the sourceHost must exist within the system
        [x] the destinationHost must exist within the system
        [x] the pair (sourceHost,desinationHost) must not be repeated
            within the system
        []
